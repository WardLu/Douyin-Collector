# 隐私政策 (Privacy Policy)

**生效日期：2026年1月6日**

"抖音采集助手"（以下简称"本扩展"）非常重视您的隐私。本隐私政策旨在说明我们如何收集、使用和保护您的信息。

## 1. 数据收集与使用

本扩展**不会**收集、存储或发送您的任何个人敏感信息到我们的服务器。

*   **用户配置信息**：您填写的飞书 App ID、App Secret 等配置信息仅保存在您的浏览器本地 (`chrome.storage.sync`)，用于与飞书 API 进行通信。
*   **网页数据**：本扩展仅在您点击采集按钮时，读取当前抖音网页上的公开可见信息（如视频标题、点赞数等），并直接发送到您配置的飞书表格中。
*   **数据流转**：数据流向仅为：您的浏览器 -> 飞书 API。数据不经过任何第三方服务器。

## 2. 权限说明

*   **activeTab**：用于检测您当前是否处于抖音页面。
*   **storage**：用于保存您的插件配置。
*   **host_permissions**：
    *   `douyin.com`：用于读取视频数据。
    *   `feishu.cn`：用于将数据保存到您的飞书表格。

## 3. 第三方服务

本扩展的功能依赖于飞书 API。在使用保存功能时，需遵循飞书的相关隐私政策。
