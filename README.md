# 抖音采集助手 - Chrome 扩展

筛选抖音博主高赞视频，一键保存到飞书多维表格。

---

## 🚀 安装方法

### 1. 加载扩展

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 打开右上角的 **「开发者模式」** 开关
3. 点击 **「加载已解压的扩展程序」**
4. 选择 `douyin-collector` 文件夹

### 2. 配置飞书信息

扩展已预设默认配置。如需修改，点击扩展图标 → ⚙️ 设置：

| 配置项 | 默认值 |
|--------|--------|
| App ID | `[YOUR_APP_ID]` |
| App Secret | `[YOUR_APP_SECRET]` |
| Base ID | `[YOUR_BASE_ID]` |
| Table ID | `[YOUR_TABLE_ID]` |

### 3. 添加文档应用权限

**重要**：首次使用前需在飞书表格中添加应用权限：
1. 打开飞书多维表格
2. 点击右上角 `...` → `更多` → `添加文档应用`
3. 搜索并添加应用

---

## 📖 使用方法

1. 访问抖音博主主页（如 `https://www.douyin.com/user/xxx`）
2. **滚动页面**加载更多视频
3. 点击扩展图标
4. 输入最低点赞数筛选条件（可选）
5. 点击 **「🔍 开始采集」**
6. 点击 **「💾 保存到飞书表格」**

---

## 📋 飞书表格字段要求

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 视频标题 | 文本 | 视频描述文案 |
| 视频链接 | 链接 | 视频详情页 URL |
| 博主名称 | 文本 | 作者昵称 |
| 博主主页 | 链接 | 作者主页 URL |
| 播放数 | 数字 | 播放量 |
| 点赞数 | 数字 | 点赞量 |
| 评论数 | 数字 | 评论量 |
| 收藏数 | 数字 | 收藏量 |
| 转发数 | 数字 | 分享量 |
| 采集时间 | 日期 | 数据采集时间戳 |

## 📁 文件结构

```
douyin-collector/
├── manifest.json      # 扩展配置
├── content.js         # 页面数据提取
├── background.js      # 飞书 API 调用（批量创建）
├── popup.html/js/css  # 操作面板
├── options.html/js/css# 设置页面
└── icons/             # 扩展图标
```

---

## ⚠️ 注意事项

1. **需要手动滚动**：插件只能采集当前页面已加载的视频
2. **页面结构变化**：抖音可能更新页面结构导致采集失败
3. **数据精度**：部分数据为简写形式（如"10.5万"）会自动转换
4. **仅供学习研究**：请遵守抖音平台使用条款

---

## 🔧 技术说明

- **框架**：Chrome Extension Manifest V3
- **兼容性**：Chrome 88+
- **数据提取**：DOM 解析
- **API调用**：飞书开放平台 Bitable API（批量创建）
