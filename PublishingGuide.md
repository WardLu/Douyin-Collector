# Chrome 商店发布指南

本指南将指导你将「抖音采集助手」发布到 Chrome Web Store。

---

## 1. 准备工作

在开始之前，请准备好以下素材：

### 📁 代码包
1. 进入 `f:\program\feishuDemo` 目录。
2. 将 `douyin-collector` 文件夹打包成一个 **ZIP 压缩包**（例如 `douyin-collector.zip`）。
   * *注意：是压缩文件夹里面的内容，还是压缩文件夹本身都可以，Chrome 都能识别。*

### 🖼️ 宣传图片（商店必填）
Chrome 商店极其严格，必须提供特定尺寸的图片：
1. **应用图标**：128x128 像素的 PNG（代码目录里 `icons/icon128.png` 即可）。
2. **应用截图**：至少 1 张，建议 1280x800 或 640x400 像素。
   * *建议截图：插件在抖音页面打开的样子，以及保存成功后的飞书表格样子。*
3. **宣传图（小）**：440x280 像素。
4. **宣传图（大）**：920x680 像素。
   * *可以使用 Canva 或 PPT 简单制作，放上插件名字和一句Slogan。*

---

## 2. 注册开发者账号

1. 访问 [Chrome 开发者信息中心](https://chrome.google.com/webstore/dev/dashboard)。
2. 使用 Google 账号登录。
3. 如果是第一次使用，需要支付 **$5 美元** 的一次性注册费（只需付一次，以后发多少插件都免费）。

---

## 3. 上传与填写信息

1. 在开发者后台首页，点击 **「+ 新增商品」**。
2. 上传刚才打包好的 `douyin-collector.zip` 文件。
3. 上传成功后，进入信息填写页面：

### 📝 商店列表
*   **名称**：抖音采集助手（会自动从 manifest 填入，可修改）
*   **说明**：复制 `README.md` 中的简介，重点突出功能。
*   **类别**：建议选“生产工具”或“搜索工具”。
*   **语言**：中文（简体）。
*   **图像资源**：上传上面准备好的图标、截图和宣传图。

### 🔒 隐私权实务
*   **单一用途**：填写“帮助用户筛选和采集抖音视频数据到飞书”。
*   **权限说明**：
    *   `activeTab`：用于识别当前是否为抖音页面。
    *   `storage`：用于保存用户的飞书配置信息。
    *   `alarms`：用于保持后台服务活跃，防止任务中断。
    *   *远程主机权限* (`douyin.com`, `feishu.cn`)：用于采集数据和调用API。
*   **数据使用情况**：
    *   是否收集用户数据？ -> **否**（数据只保存在本地或用户自己的飞书里）。
    *   由于我们没有后端服务器，这里尽量全选“不收集/不使用”。

### 🌐 隐私政策 URL
*   你需要提供一个隐私政策的链接。
*   **简单做法**：将我生成的 `PRIVACY.md` 内容复制，发布到 GitHub Gist、Notion 公开页或飞书文档公开页，然后把那个公开链接填进去。

---

## 4. 提交审核

1. 所有必填项填完后，点击右上角的 **「提交审核」**。
2. **审核时间**：通常需要 1-3 个工作日。
3. 审核通过后，你会收到邮件通知，插件就会在 Chrome 商店上线了！

---

## 💡 商业化设置（后续）
如果你以后想开启付费订阅：
1. 在开发者后台，不需要改代码。
2. 可以在“分发”选项卡中设置“付费”状态（注意：Chrome 官方支付已停用，现在通常需要自己接入 Stripe/微信/支付宝，就像我们在代码里做的打赏页那样）。
